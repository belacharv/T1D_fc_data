# T1D_fc_data

This repository is part of diplomma thesis Characterization of T cells in children with type 1 diabetes mellitus. The scripts uploaded above are purposed for flow cytometry analysis using both manually gated population and unsupervised clustering. The unsupervised clustering is perfomed using Phenograph embeded in framework cyCONDOR on the R background. The observed batch effect in our data was handled using Harmony algorithm. The pipeline can be used for analysis of any fc data, although it needs to be adjusted (for example annotation tables, the directory with data etc.)
